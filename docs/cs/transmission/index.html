    vsftpd

    Very Secure FTP Daemon。

    只允许某些用户访问[2]

    /etc/vsftpd.conf:

    userlist_enable=YES
userlist_file=/etc/vsftpd.userlist
userlist_deny=NO

    INVALIDARGUMENT[3]

    /etc/vsftpd.conf:

    listen=YES

    530 Login incorrect[4]

    /etc/vsftpd.conf:

    pam_service_name=ftp
