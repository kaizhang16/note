    线性代数

    相似矩阵

    • 同一个线性变换在不同基下的矩阵称为相似矩阵

    正交矩阵[1]

    \[\transpose{\matrix{Q}}\matrix{Q} = \matrix{Q}\transpose{\matrix{Q}} = \matrix{I}\qquad(1)\]

    旋转矩阵[2]

    旋转矩阵的集合：

    \[ \SO(n) = \{\matrix{R} \in \set{R}^{n\times n} \mid \matrix{R}\transpose{\matrix{R}} = \matrix{I}, \det(\matrix{R}) = 1\} \qquad(2)\]

    外积[3]

    \[\vec{u} \otimes \vec{v} = \vec{u}\transpose{\vec{v}}\qquad(3)\]
