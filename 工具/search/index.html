    搜索

    1 搜索命令

    1.1 navi

    1.1.1 安装

    NAVI_VERSION=2.4.1
curl -LO "https://github.com/denisidoro/navi/releases/download/v$NAVI_VERSION/navi-v$NAVI_VERSION-x86_64-unknown-linux-musl.tar.gz"
tar -xzvf navi-v$NAVI_VERSION-x86_64-unknown-linux-musl.tar.gz
mv -i navi ~/.local/bin/

    1.1.2 使用

    Ctrl-g。
